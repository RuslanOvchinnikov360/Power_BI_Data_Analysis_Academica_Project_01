# Power BI E-Commerce Analytics Dashboard
This project demonstrates the development of an end-to-end business intelligence solution in Power BI using data from a MySQL database.

The report includes:
* Sales and profitability analysis
* Customer and regional performance analysis
* Warehouse and logistics performance monitoring
* Employee efficiency evaluation
* KPI tracking using DAX measures
* Interactive dashboards with filters and drill-down capabilities

Key metrics developed include Total Sales, Cost, Profit, Profit Margin, Average Order Value (AOV), Returns Rate, Warehouse Revenue, and Revenue per Employee.

The objective of the project was to transform raw transactional data into meaningful business insights, identify sales trends, evaluate operational performance, and support data-driven decision-making through effective visualization and analytics.
